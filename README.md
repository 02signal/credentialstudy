# CredentialStudy

`credentialstudy.com` — the online learning / provider brand of the Ettevõtluskeskus OÜ /
02Signal site group. An online learning brand for focused, recognised microcredentials for
working adults across Europe.

This is the **provider/academy** brand. The neutral microcredentials register is a
separate brand and repo (`mikrokvalifikatsioon.ee`).

## Stack
Astro static site (Vercel + Cloudflare DNS), same family pattern.

## Develop
```bash
npm install
npm run dev      # local dev
npm run build    # astro check + build
npm run preview  # serve dist/
```
